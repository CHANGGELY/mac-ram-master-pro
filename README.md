# 🚀 RAM Master | Mac 运存清理大师 (永久免费版)

一款具有 **Apple 级审美** 的开源 macOS 运存优化工具。专为极致体验设计，一键清空不活跃缓存，让你的 Mac 瞬间回春。

---

## ✨ 核心亮点 (Key Features)

- **Apple 级 UI/UX 体验**：
  - **极致审美**：采用深空灰 (Space Gray) 基调，配合翡翠绿 (Emerald Green) HSL 配色。
  - **毛玻璃效果**：深度定制的 `backdrop-filter` 面板，完美契合 macOS 系统质感。
  - **灵动动画**：基于 `Framer Motion` 的物理引擎级过渡，每一帧都丝滑顺畅。
- **底层精准优化**：
  - **智能解析**：通过正则表达式实时解析 `vm_stat` 原始页数据，兼容多版本 macOS 系统。
  - **原生清理**：调用苹果官方 `purge` 内核指令，强制回收不活跃 (Inactive) 内存，真实释放空间。
- **SOTA 架构设计**：
  - **中文化编程**：代码逻辑采用全中文命名与注释，降低初学者理解成本。
  - **隔离沙箱**：主进程与渲染进程通过 `ContextBridge` 严格隔离，杜绝安全风险。
  - **零容忍报错**：全量 TypeScript 覆盖，严谨的类型推断。

## 📦 开发者快速上手 (Quick Start)

### 1. 环境准备
确保你已安装 `Node.js` (推荐 v18+)

### 2. 初始化项目
```bash
# 安装依赖
npm install
```

### 3. 开发环境运行
```bash
# 启动 Vite 服务与 Electron
npm run dev
# 在另一个终端启动
npm run electron:dev
```

### 4. 构建发布
```bash
# 生成 macOS 安装包
npm run dist
```

## 👨‍🏫 编程导师笔记 (Mentor Notes)

> **什么是 Inactive Memory (不活跃内存)？**
> 想象你的桌面上堆满了之前看过的书（缓存），虽然你现在没在看，但它们占着地方。当你需要摊开一张大地图时，`purge` 指令就像是一位管家，迅速把这些没用的书收回书架，把干净的桌面（物理内存）腾出来给你的新任务。
